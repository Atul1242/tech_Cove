⚙️ Implementation Rules for AI

Use Vite + React project structure.

Bootstrap 5 via npm (import 'bootstrap/dist/css/bootstrap.min.css').

Create reusable components: Navbar, Footer, Hero, PricingCard, LocationCard, TestimonialCarousel, etc.

Use React Router for page navigation.

Store dummy data in src/data/*.json or TS objects.

Keep backend-like logic as stubs (e.g., function handleBooking() { /* TODO: connect API */ }).

For PWA readiness: include manifest + service worker placeholder.

Code should be modular, easy to expand, and deployable to Netlify/GitHub Pages without modification.
